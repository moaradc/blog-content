本文用于测试博客的代码块渲染逻辑、复制功能以及折叠/展开功能。

### 1. JavaScript 基础示例

这是最基础的 JS 代码块，使用 `language-js`。

```js

// 欢迎来到控制台

console.log('Hello, Moara Blog!');

const add = (a, b) => {

    return a + b;

}

alert(add(10, 20));

```

### 2. HTML 结构示例

测试 HTML 高亮显示。

```html

<!DOCTYPE html>

<html>

  <head>

    <title>测试页面</title>

  </head>

  <body>

    <div class='container'>

      <h1>Hello World</h1>

    </div>

  </body>

</html>

```

### 3. 长代码测试 (测试展开功能)

当代码较长时，应显示滚动条或通过 CSS 限制高度（配合 `expand-btn` 逻辑）。

```css

/* 全局重置 */

* {

    margin: 0;

    padding: 0;

    box-sizing: border-box;

}

body {

    background-color: #f5f5f5;

    font-family: 'Noto Sans SC', sans-serif;

    line-height: 1.6;

    color: #333;

}

.header {

    background: #222;

    color: #fff;

    padding: 20px;

    text-align: center;

}

.nav-link {

    color: #fff;

    text-decoration: none;

    margin: 0 10px;

}

.nav-link:hover {

    color: #00adb5;

}

/* 底部样式 */

footer {

    margin-top: 50px;

    padding: 20px;

    border-top: 1px solid #ddd;

    text-align: center;

    font-size: 0.9rem;

}

```

### 4. 无语言标记 (Plaintext)

## 什么是Markdown？

Markdown是一种轻量级标记语言，让你可以使用纯文本格式编写文档，然后转换成有效的HTML。

## 基本语法

### 文本格式

- **粗体文本**使用双星号
- *斜体文本*使用单星号
- ~~删除线~~使用双波浪号

### 代码

行内代码：`console.log('Hello')`

代码块：

```javascript
function greet(name) {
    return `Hello, ${name}!`;
}
```
